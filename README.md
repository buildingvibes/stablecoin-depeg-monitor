# StableGuard - Stablecoin Depeg Monitor

Real-time monitoring of stablecoin stability, depeg risks, and reserve health for major USD-pegged stablecoins.

**Live Dashboard**: [https://buildingvibes.github.io/stablecoin-depeg-monitor/](https://buildingvibes.github.io/stablecoin-depeg-monitor/)

## What It Does

StableGuard tracks 10 major USD stablecoins and provides:

- **Peg Deviation Tracking**: Real-time price monitoring with deviation from $1.00 peg
- **Risk Scoring**: Composite risk scores (0-100) based on price deviation, 7-day volatility, and market capitalization
- **7-Day Price Charts**: Visual price history showing stability patterns
- **Event Monitoring**: Stablecoin-related incidents (depeg events, reserve concerns, regulatory actions, security incidents) via CPW Tracker API
- **AI Risk Analysis**: Weekly intelligence briefing generated by GitHub Models (GPT-4o-mini)
- **Automated Updates**: GitHub Actions fetches fresh data every Sunday

## Tracked Stablecoins

USDT (Tether), USDC (USD Coin), DAI, FDUSD, FRAX, TUSD, PYUSD (PayPal), USDe (Ethena), USDD, GUSD (Gemini)

## Data Sources

- **[CPW Tracker API](https://rapidapi.com/CPWatch/api/cpw-tracker)**: Stablecoin-related events (depeg, reserve, regulation, cyberattack)
- **[CoinGecko API](https://www.coingecko.com/)**: Real-time prices, market caps, 24h volumes, 7-day price history
- **[GitHub Models](https://docs.github.com/en/github-models)**: AI-powered weekly risk analysis

## Risk Score Methodology

Each stablecoin receives a risk score from 0-100 based on:

| Factor | Max Points | Description |
|--------|-----------|-------------|
| Peg Deviation | 40 | Maximum deviation from $1.00 over 7 days |
| Volatility | 30 | Standard deviation of price changes over 7 days |
| Market Cap | 30 | Lower market cap = higher risk (less liquidity) |

Risk levels: **LOW** (0-19) | **MEDIUM** (20-39) | **HIGH** (40-59) | **CRITICAL** (60-100)

## Setup

1. Clone this repository
2. Subscribe to [CPW API](https://rapidapi.com/CPWatch/api/cpw-tracker) (Basic plan, 100 free requests/month)
3. Add `RAPIDAPI_KEY` to repository secrets
4. Enable GitHub Pages (Settings > Pages > Source: GitHub Actions)
5. Run the workflow manually or wait for the Sunday schedule

## Local Development

```bash
# Fetch data (RAPIDAPI_KEY optional - CoinGecko data works without it)
node scripts/api-call.js

# Generate AI analysis (requires GITHUB_TOKEN)
GITHUB_TOKEN=your_token node scripts/ai-analysis.js

# Serve locally
npx serve .
```
